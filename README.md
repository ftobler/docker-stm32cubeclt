
# STM32CubeCLT in Docker

[![build_docker](https://github.com/ftobler/docker-stm32cubeclt/actions/workflows/build_docker.yml/badge.svg)](https://github.com/ftobler/docker-stm32cubeclt/actions/workflows/build_docker.yml)

This project provides a Docker image with [STM32CubeCLT](https://www.st.com/en/development-tools/stm32cubeclt.html) installed, based on Ubuntu and Node.js.

[ftobler/stm32cubeclt](https://hub.docker.com/r/flotobler/stm32cubeclt)

## Features

- Ubuntu base image
- STM32CubeCLT pre-installed
- Node.js included
